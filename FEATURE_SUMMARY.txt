│                          📊 NEW API ENDPOINTS                                │
└──────────────────────────────────────────────────────────────────────────────┘

   GET /api/fees/receipt/:receiptNumber
   ├─ Downloads fee receipt PDF
   ├─ Requires JWT authentication
   └─ Returns: PDF file (receipt-REC00001.pdf)

   GET /api/admissions/certificate/:studentId
   ├─ Downloads admission certificate PDF
   ├─ Requires JWT authentication
   └─ Returns: PDF file (admission-STU001.pdf)

│                          📞 TROUBLESHOOTING                                  │
└──────────────────────────────────────────────────────────────────────────────┘

   Problem: PDF not downloading
   Solution: Run "npm install" and restart server

   Problem: "pdfkit not found" error
   Solution: cd to project root, run "npm install pdfkit"

   Problem: Download button not visible
   Solution: Clear browser cache, restart frontend

   Problem: PDF is blank
   Solution: Check server console for errors
